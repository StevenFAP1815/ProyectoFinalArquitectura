# Pre-requisitos para usar el sistema
Se utilizó modo desarrollo de excel, lo escencial es activar el modo desarrollo en configuraciones, a continuación se presenta el paso a paso:

En Windows (Excel 2016, 2019, 2021, Microsoft 365)

- Abre Excel.
- Haz clic en Archivo (esquina superior izquierda).
- Selecciona Opciones (al final del menú).
- En la ventana que se abre, entra en Personalizar cinta de opciones.
- En la columna derecha, busca y marca la casilla “Desarrollador”.
- Haz clic en Aceptar.

En Mac (Excel para macOS)

- Abre Excel.
- En la barra superior, haz clic en Excel → Preferencias.
- Selecciona Cinta de opciones y barra de herramientas.
- En la lista de pestañas principales, marca “Desarrollador”.
- Cierra la ventana.
